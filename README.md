# Pow

A Flutter app to manage customers, inventory and billings of a swimming centre, Samarahan Power Penguin Swimming Club. It can run on Android of version 5 and above. It uses Firebase Cloud Firestore as database.

</br> 

## Understanding Pow

The features of this app are as follows: 
 
● Log In 
 
● Manage Billing 
 - Add billing 
 - View billing 
 
● Manage Inventory 
 - Add item 
 - View item 
 - Edit item 
 - Delete item 
 
● Manage Customers 
 - Add customer 
 - View customer 
 - Edit customer 
 - Delete customer 
 
● Log Out

</br> 

## Installing Pow

Download this [apk](https://github.com/ariessa/PowApp/blob/old/build/app/outputs/apk/app.apk) into your Android device and install the app.
